Reaction Handler

Handles reactions for the tag system.

For more information about the tag system, see this page.

Trigger#

Type: Reaction
Additional options: Added reactions only

{{/*	Manages pagination for the tag system.	See <https://yagpdb-cc.github.io/tags/reaction-handler> for more information.
