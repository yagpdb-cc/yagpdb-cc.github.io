Main CC

This command allows users to set an AFK message with optional duration.

For more information about the AFK system, please see this page.

Trigger#

Type: Regex
Trigger: \A

Usage#

  • -afk <message> - To set an AFK message with no expiration.
  • -afk <message> -d <duration> - To set an AFK message that expires in duration time.

Configuration#

See the AFK system overview for instructions regarding how to configure this command.
